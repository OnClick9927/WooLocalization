# WooLocalization [Doc](https://onclick9927.github.io/2024/10/26/Doc/WooLocalization/1-WooLocalization-%E7%AE%80%E4%BB%8B)

*  支持  文本/图片/特效/预制体/继承自UnityEngine.Object 本地化
*  提供本地化组件（支持挖孔的文本，支持多种属性变化）
*  支持一键翻译 
*  支持无限多语言类型
*  支持扩展自定义本地化
*  支持导入导出自定义本地化文件
